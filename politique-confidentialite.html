Informations sur la Confidentialité et l'Utilisation des Données

Dernière mise à jour : [Date de la dernière mise à jour]

Bienvenue sur EmploiAvenir. Votre confiance est importante pour nous. Cette page explique comment nous traitons vos informations personnelles et comment l'intelligence artificielle (IA) est utilisée sur notre site.

1. Conservation de vos Données Personnelles

Stockage des données : Actuellement, EmploiAvenir est conçu pour fonctionner principalement avec les données stockées localement dans votre navigateur via la technologie `localStorage`. Cela signifie que les informations que vous générez (progression dans le parcours, simulations de candidatures, documents créés, etc.) sont sauvegardées directement sur votre appareil et ne sont pas envoyées à nos serveurs.

  • Ce qui est stocké localement : Votre progression dans "Mon Parcours", les informations des formulaires que vous remplissez (comme les candidatures simulées, les informations pour la génération de CV/LM si applicable), et vos préférences (comme la langue).
  • Ce qui n'est PAS stocké (actuellement) : Nous ne collectons ni ne stockons d'informations personnelles identifiables sur des serveurs distants sans votre consentement explicite pour des fonctionnalités spécifiques qui pourraient être ajoutées à l'avenir (par exemple, la création d'un compte utilisateur avec identifiants).

Durée de conservation : Les données stockées localement dans votre navigateur y restent tant que vous ne videz pas le cache de votre navigateur pour ce site, ou tant que vous n'utilisez pas une fonctionnalité explicite de suppression de données si elle est proposée.

Vos droits : Puisque les données sont stockées localement :

  • Droit d'accès et de rectification : Vous pouvez accéder et modifier vos informations directement via les fonctionnalités du site (par exemple, modifier une candidature dans "Mes Candidatures").
  • Droit à l'effacement ("droit à l'oubli") : Vous pouvez effacer toutes les données stockées par EmploiAvenir en vidant les données de site de votre navigateur (cache et stockage local). Si une fonctionnalité de "Supprimer mon compte" ou "Réinitialiser mes données" est ajoutée, elle effectuera cette action.
  • Droit à la portabilité : Certaines données, comme les documents générés, peuvent être téléchargées.

Sécurité : Bien que les données soient locales, nous vous encourageons à sécuriser l'accès à votre navigateur et à votre appareil.

Évolution future : Si EmploiAvenir évolue pour inclure un système de comptes utilisateurs avec stockage de données sur serveur, cette politique sera mise à jour et votre consentement sera requis pour de tels traitements.
